## 🚀 Quick Start

### 1. Requirements
Ensure you have the quantum libraries installed:
```bash
pip install qiskit-aer qiskit fastapi uvicorn moviepy opencv-python librosa yt-dlp
```

### 2. Launch Backend
```bash
cd backend
python -m uvicorn main:app --port 8000
```

### 3. Launch Frontend
```bash
cd frontend
npm run dev -- --port 5174
```
